# Kamran Khan — Textile Designer Portfolio

Professional portfolio website for **Kamran Khan**, Textile & Carpet Designer.

## Quick Start

Open `index.html` in any modern web browser, or run:

```bash
npx serve .
```

Then visit `http://localhost:3000`.

## PDF Version

1. Open `index.html` in Chrome or Edge.
2. Press **Ctrl+P** (Print).
3. Choose **Save as PDF**.
4. Enable **Background graphics**.

## Files

| File | Purpose |
|------|---------|
| `index.html` | Full portfolio (all sections) |
| `css/styles.css` | Premium responsive styling |
| `js/main.js` | Navigation, lightbox, scroll effects |
| `assets/images/` | Organized design samples |
| `PORTFOLIO_SUMMARY_REPORT.md` | Images used, projects, gaps, recommendations |
